# 今日收获

## 阅读总结

### 填空题核心流程

1. 看字数限制
2. 阅读题干
3. 预测词性
4. 找定位词
5. 识别同义替换
6. 回原文定位答案

一句话总结：

> 填空题本质 = 定位词 + 词性预测 + 同义替换

---

### TFNG核心判断逻辑

TRUE

* 原文信息一致
* 可以改写

FALSE

* 原文信息冲突
* 出现反义关系

NOT GIVEN

* 找不到对应信息
* 找得到单词但找不到关系

一句话总结：

> TFNG判断的是关系，而不是单词。

---
