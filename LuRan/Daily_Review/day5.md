### 阅读
> 记住：题目抽象文章具体

* 第六章：模拟训练（三节）
  > C11 - T1 - P1 2 3
  
  一.P1：填空 + 判断（**送分**）
  - 1、先根据填空题做法（**day1**）*题文同序、题抽文具* 逐一解决掉填空（填的都是原文），同时读通全文，标注关键词
  - 2、再做判断题（**day1**），此时效率很高因为已经读过文章
  
  二.P2：示意图（本质填空） + 判断
  - 1、看示意图的时候，要看清楚图周围的**辅助信息句子，小细节**，作为关键词去原文中匹配，**填空信息段不一定在文章开头，记得先定位答案段落**
  - 2、填词的时候填`n`，别把前面的`adj`也扔进来了，注意填词数量上限，以及单复数（a pair of sth = two sths，**填sth无s**）
  - 3、判断题，记得**翻译一下题干大意意思**，**找 4+1 定位词**，这样在做填空浏览文章的时候能有点印象，重点关注`only`这种数量、绝对性的词，很容易触发**NO/NG**

  三.P3：段落匹配 + 表格填空 + 人名类匹配
  - 1、段落匹配不一定题文同序，关注题目中**抽象**的`n`去找文章段落中的对应**具体**部分匹配
  - 2、人名类匹配（其NB是真NB）：*先* 根据选项**自上而下**找出人名对应段落，*再* 读懂题干，*最后* 用题干与文本对应

## 今日收获总结
 - 套卷总结：
   ```
   1. 答题情况：阅读：P1错 1 题；P2错 2 题；P3错 4 题
   2. 错误原因：P1-12，错在没看清文本，只匹配了关键词没有搞清定位句的表达逻辑；P2-14，错在定位失误；P2-23，题抽文具，没看懂文本；P3段落匹配和人名匹配没做好，本质阅读记忆力差，需要进一步练习
   ```
 - 套卷策略&答题技巧：
   - **拿到阅读先看下题型决定做题策略**，其中填空题真的很简单，各种匹配题要注意关键词解析
   - 同意替换常常是 *动词* 和 *方位词* 如`keep`和`remain`、`beneath`和`under`
   - 整个阅读Test，**P1很简单，基本都是`填空/匹配+判断`**;**P2也简单，基本都是`填空+判断`**;**P3稍难，会有`选择+匹配+填空`等多题型**
   - 多听多说有关**日期、数字、钱**的口语表达
   - 
 - 积累：
   - `A of B` = B的A
   - `most conservative estimate` = 最保守的估计
   - `cutting-edge` = 尖端的，边缘优势
   - `scale up` = 扩大
   - `horizontal/vertical` = 水平的/垂直的
   - `subject sth` = 放置
   - `eliminate the need for` = 消除了对xxx的需要
   - `rooftops` = 屋顶
   - `painstakingly` = `assiduous` = `diligent` = 刻苦勤勉
   - `diametrically` = 完全地，作为直径的，相反的
   - `axle` = `axis` = 轴
   - `in this regard` = 在这种方面
   - `let alone doing sth` = 更不必说
   - `sth has been shown to work` = 已经发生了
   - `exploit/deposit` = 开采/放置积存沉淀
